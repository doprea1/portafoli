Parrafo con Estilo

Este es un parrafo de ejemplo donde algunas palabras estan resaltadas para mostrar el efecto subrayado. Puedes usar la etiqueta strong para enfatizar palabras importes. El fondo de estas palabras es amarillo semitransparente, lo que ayuda a que se destaquen en el texto. ¡Prueba a añadir mas texto y ver como se ve!
